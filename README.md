# Kaiju Breakdown
A Cozy Kaiju Cleanup Game

# Initial Dev Setup

1. Clone this repo
2. Install the latest version of the Godot Editor from https://godotengine.org/download
3. Run `setup.command` (explorer) or `setup.sh` (terminal) to install dependencies (MacOS only currently)
4. Open the `project.godot` file with your Godot Editor
5. Make some game
