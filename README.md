# Kaiju Breakdown
A Cozy Kaiju Cleanup Game

Engine: **Godot 4.6 (.NET / C#)**. This is a C# project (`KaijuBreakdown.sln` / `KaijuBreakdown.csproj`); use the **.NET / Mono** build of the Godot editor.

# Initial Dev Setup

1. Clone this repo
2. Install the **.NET (Mono) build** of the Godot 4.6 editor from https://godotengine.org/download
3. Install the **.NET SDK 8.0** (or newer) from https://dotnet.microsoft.com/download
4. Run `setup.command` (explorer) or `setup.sh` (terminal) to install Godot addon dependencies (MacOS only currently)
5. Open the `project.godot` file with your Godot Editor (the editor generates the C# build files on first open)
6. Make some game

## Build

- Build the C# project: `dotnet build` (or build from the Godot editor)
- Run: open the project in the Godot 4.6 .NET editor, or `godot --path .`
